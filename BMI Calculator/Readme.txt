CPSC 411 - Assignment 4 BMI Calculator
CWID 893925362

BMI Calculator: This program will take the height and weight of a user and it will calculate his/her BMI

How to run the program:
1. Open the file Assignment1.xcodeproj
2. All the supporting files should be there
3. Run the application
4. Input the value wanted for height and weight
5. Click the switch button to change from US measurement to NonUS measurement (By default it is us measurement)
6. Click go and the program will output the result.

All the features have been implemented. 
Pictures are created using paint
